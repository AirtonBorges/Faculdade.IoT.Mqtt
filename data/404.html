Gerenciar arquivos
Se você estiver configurando a interface, envie os arquivos para o dispositivo usando uma das rotas abaixo:
  • /upload — endpoint HTTP multipart (campo `file`) para enviar e gravar diretamente no LittleFS.
  • /fs/list — lista arquivos presentes no LittleFS (retorna JSON).
  • /fs/download?name=<path> — baixa um arquivo do LittleFS (ex: /fs/download?name=/index.html).
  • /fs/delete — apaga um arquivo; aceita POST (campo form `name`) e também GET via query string `?name=` (ex: /fs/delete?name=/index.html).
Exemplos rápidos (curl): 
curl -F "file=@index.html" http://<ESP_IP>/upload
curl http://<ESP_IP>/fs/list
curl -o index.html "http://<ESP_IP>/fs/download?name=/index.html"
curl -X POST -d "name=/index.html" http://<ESP_IP>/fs/delete
curl "http://<ESP_IP>/fs/delete?name=/index.html"
